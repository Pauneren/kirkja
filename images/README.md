# Images Folder

This folder contains images for the church website.

## Recommended Images:

### Church Images:
- `olafskirkja.jpg` - Main image for Olafskirkja church
- `ingjalsholskirkja.jpg` - Main image for Ingjalsholskirkja church
- `church-hero.jpg` - Background image for hero section (recommended: wide church interior or exterior)

### News Images:
- `community-dinner.jpg` - Community dinner event
- `youth-ministry.jpg` - Youth ministry program
- `church-renovation.jpg` - Church renovation
- `bible-study.jpg` - Bible study group
- `christmas-outreach.jpg` - Christmas outreach
- `worship-team.jpg` - Worship team

## Image Specifications:
- **Church images:** Recommended size 1200x800 pixels
- **Hero image:** Recommended size 1920x1080 pixels (wide format for background)
- **News images:** Recommended size 600x400 pixels
- Format: JPG or PNG
- Keep file sizes reasonable for web loading

## Usage:
Images are referenced in HTML files like:
```html
<img src="images/olafskirkja.jpg" alt="Olafskirkja" class="church-image">
```

## Hero Background:
The hero section uses `church-hero.jpg` as a background with a semi-transparent gradient overlay. This creates a beautiful effect where the church image shows through while maintaining text readability.
